# Contexte — Projet SportBook

## Contexte de formation
- **Formation :** Titre professionnel DWWM (Développeur Web et Web Mobile)
- **Établissement :** MolenGeek
- **Durée :** 6 mois
- **Projet :** SportBook — projet de fin de formation
- **Objectif :** Application web de réservation de séances sportives (Node.js/Express/MySQL, frontend SPA vanilla)

## Notes générales
- Le journal de bord (`docs/journal.md`) doit être tenu à jour régulièrement — il sert également de base pour remplir le dossier professionnel.
- Les entrées du journal doivent couvrir à la fois les aspects techniques ET les compétences transversales (organisation, gestion de projet, problématiques rencontrées, etc.).
- Les commits doivent être accompagnés d'une mise à jour du journal via la skill `commit-journal`.
